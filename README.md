# Transaction-Fraud-Dashboard-
Project Overview

This project presents an interactive Financial Transaction Fraud Analysis Dashboard built using Microsoft Power BI. The dashboard analyzes banking transaction data to identify fraudulent activities, monitor transaction patterns, and provide actionable insights for improving fraud detection and risk management.

The report is designed to help financial institutions understand fraud trends, identify high-risk transactions, and support data-driven decision-making through interactive visualizations.

Objectives

Analyze fraudulent and genuine banking transactions.
Identify high-risk transaction categories and payment methods.
Monitor fraud trends over time.
Compare domestic and international transaction risks.
Enable interactive analysis using filters and slicers.

Tools & Technologies

Microsoft Power BI
Power Query
DAX (Data Analysis Expressions)
Interactive Dashboard Design
